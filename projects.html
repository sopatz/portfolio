Interactive US County Map with City Populations

An interactive county map of the US made using Leaflet, a library of JavaScript specializing in interactive maps. The user can search or click on any county in the US, and the five (or user-specified number, five is the default) largest cities will appear on the map with their locations and populations. Made for my brother with Asperger's Syndrome, who is really interested in US population data and its quirks, and has been to all 88 counties in Ohio.

Tools Used:

  • JavaScript (Leaflet)
  • HTML/CSS
  • Python
  • GitHub Actions
Open View on GitHub

El Dorado Fire Analysis

Analysis of the impacts of the El Dorado fire, which was sparked by a gender reveal celebration on Sept. 5, 2020 near San Bernardino, California. The burned areas, burn severity, and vegetation loss were mapped and analyzed using Google Earth Engine. The user can click anywhere on the map in Google Earth Engine to generate a NDVI time-series chart at the specified location.

Tools Used:

  • Google Earth Engine
  • JavaScript
